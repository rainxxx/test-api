添加/修改活动

作者: guyang date 2020.07.06

请求URL

/admin/activity/insertOrUpdate POST

请求参数

{
  "operationType": "int //操作类型 1.添加 2.修改【必须】",
  "id": "long //活动Id",
  "name": "string //活动名称",
  "type": "int //活动类型 10分享 20团销 30秒杀",
  "image": "string //活动导航图",
  "isEnable": "boolean //是否已启用(上架)",
  "description": "string //活动描述(富文本)",
  "activityRewardRuleId": "long //奖励规则id",
  "spuIdList": "int[] //参数活动的商品Id列表"
}

返回结果

{
  "code": "int //返回码,0表示成功",
  "message": "string //返回提示消息",
  "data": {}
}

活动列表

作者: guyang date 2020.07.06

请求URL

/admin/activity/list GET

请求参数

参数名 类型 必须 描述
pageNo int 否 页码
pageSize int 否 每页条数
id long 否 活动Id
name string 否 活动名称[模糊匹配]
isEnable boolean 否 是否上架

返回结果

{
  "code": "int //返回码,0表示成功",
  "message": "string //返回提示消息",
  "data": {
    "total": "long //总记录数",
    "items": [
      {
        "id": "long //活动Id",
        "name": "string //活动名称",
        "type": "int //活动类型 10分享 20团销 30秒杀",
        "image": "string //活动导航图",
        "isEnable": "boolean //是否已启用(上架)",
        "activityRewardRuleId": "long //奖励规则id",
        "typeDesc": "string //活动类型枚举描述(分享/团销/秒杀...)"
      }
    ]
  }
}
